# yulcript
## 소개
visual studio code를 사용한 스크립트 언어 지원 확장을 개발하는 간단한 레포지토리입니다.

yulcript는 기초적인 수준의 데이터 저장, 논리 구조를 지원하는 간단한 스크립트 언어입니다. 이런 가벼운 언어를 지원하는 방법을 커밋 별로 보여준는 것이 이 프로젝트의 목적입니다.

## 시작하기
- vscode v1.53.0 이상
- npm v6.14.8 이상

## yulcript
언어 확장 지원을 위한 가벼운 스크립트 언어입니다.
C 언어와 유사한 변수 선언방식, 사칙연산과 논리 연산(and, or), 조건문과 반복문을 지원합니다.
